== Description ==

This plugin integrates a banner of ads.zyntux.com in your WordPress and gives you the opportunity for advertise within our free adservices. An account on zyntux.com set is not necessary.

Just go to the <a href="http://www.zyntux.com/bannertausch/" target="_blank">Form</a>, transmits the relevant data and after maximal 48 hours, the banner will appear in our system and your WordPress installation.

== Beschreibung ==

Deutsch: Dieses Plugin bindet einen Werbebanner von ads.zyntux.com in euer Wordpress ein und gibt euch dafür selbst die Moeglichkeit kostenlos innerhalb des unseres Adservices zu werben. Ein Account auf zyntux.com wird dafür nicht benötigt. 

Geht einfach zu diesem <a href="http://www.zyntux.com/bannertausch/" target="_blank">Formular</a>, tragt die relevanten Daten ein und spaetestens nach 48 Stunden erscheint der Banner in unserem System bzw. eurer Wordpress Installation.

== Installation ==

1. Upload `zynads` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
